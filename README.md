
## Setup Environment - Terminal Ubuntu

```
python3 -m venv .env
source .env/bin/activate
pip install pandas matplotlib seaborn babel numpy streamlit
```

## Run Streamlit App

```
streamlit run dashboard/dashboard.py
```
